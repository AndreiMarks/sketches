Boxit
Dropbox integration in your Unity Apps


Before you can use Boxit, you will need to have a Dropbox
account, and at least one Dropbox app set up in that account.
You can set these up at:

http://www.dropbox.com

Please read through the user manual in the Documentation folder
for a complete walkthrough on how to set up your Dropbox
account, create an app, and use Boxit in Unity.
